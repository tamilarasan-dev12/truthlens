API Docs

Prediction Endpoint

POST

/api/v1/predict

Submit either raw statement text or a source URL. TruthLens resolves input, scores the claim, and stores the result in history.

  • Content-Type: application/json
  • Accepted keys: text, url
  • Response: status envelope with prediction payload

Example Request

curl -X POST http://localhost:5000/api/v1/predict ^
  -H "Content-Type: application/json" ^
  -d "{\"text\": \"A viral post claims a city banned tap water after a meteor impact.\"}"

Example Response

{
  "status": "success",
  "data": {
    "prediction": "Fake",
    "confidence": 0.91,
    "probabilities": {
      "Fake": 0.91,
      "True": 0.09
    },
    "analyzed_text": "..."
  },
  "error": null
}

Validation Notes

  • Text input must contain at least 20 characters.
  • URLs must begin with http:// or https://.
  • Remote page content is extracted before analysis.
  • Successful predictions are automatically appended to history.
